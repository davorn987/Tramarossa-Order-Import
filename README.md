# Tramarossa-Order-Import
Import Ordini Stagionali Tramarossa

## Configurazione
Le credenziali API e del database devono essere fornite tramite il file locale
`~/.clientimport.properties` oppure tramite variabili d'ambiente (`WC_CONSUMER_KEY`,
`WC_CONSUMER_SECRET`, `GF_CONSUMER_KEY`, `GF_CONSUMER_SECRET`, `DB_USER`,
`DB_PASSWORD`). Non salvare queste informazioni nel repository.
